Programação orientada a objeto

Sobre Conceitos básicos de orientação a objeto, aprendi que é uma técnica de programação que permite modelar objetos do mundo real em um sistema de software. A Abstração é um dos pilares da POO e é utilizada para definir entidades do mundo real. Encapsulamento é a técnica utilizada para esconder uma ideia, ou seja, não expor detalhes internos para o usuário, tornando partes do sistema mais independentes possível. Herança é um conceito que permite criar uma nova classe a partir de uma classe existente. Polimorfismo é a capacidade de um objeto poder ser referenciado de várias formas.

Sobre Orientação a objetos em java, aprendi que é uma linguagem de programação orientada a objetos. Alguns dos métodos, condições, estruturas de repetição e operadores em java que aprendi são: if, else, switch, while, do-while, for, break, continue, return, ==, !=, >, <, >=, <=, &&, ||, +, -, *, /, %, ++, --, +=, -=, *=, /=, %=.
