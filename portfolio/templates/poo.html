Programação Orientada a Objeto

Eu estudei vários conteúdos interessantes sobre programação orientada a objetos (POO) neste semestre. A seguir, vou resumir alguns deles.

POO é um paradigma de programação que trata os dados e as operações como objetos, que são entidades que possuem atributos e métodos. A POO serve para facilitar o desenvolvimento, a manutenção e a reutilização de software, pois permite abstrair, modularizar e organizar melhor o código.

Abstração, polimorfismo, encapsulamento e herança são conceitos fundamentais da POO. Eles são:

Abstração: é a capacidade de representar as características essenciais de uma entidade, ignorando os detalhes irrelevantes. A abstração permite simplificar e generalizar os problemas e as soluções.

Polimorfismo: é a capacidade de assumir diferentes formas ou comportamentos, dependendo do contexto. O polimorfismo permite que um mesmo nome ou símbolo se refira a diferentes objetos ou métodos, que compartilham uma interface comum.

Encapsulamento: é a capacidade de ocultar os detalhes internos de um objeto, expondo apenas os serviços que ele oferece. O encapsulamento permite proteger e controlar o acesso aos dados e aos métodos de um objeto, garantindo a sua integridade e a sua segurança.

Herança: é a capacidade de criar novos objetos a partir de objetos existentes, aproveitando as suas características e os seus comportamentos. A herança permite estabelecer relações de especialização e generalização entre os objetos, evitando a repetição e a redundância de código.

Para estudarmos os conceitos básicos de POO nesse semestre a linguagem utilizada foi Java, a seguir vou adentrar em alguns conceitos básicos e importantes da linguagem

Variáveis: em Java, as variáveis devem ser declaradas com um tipo e um nome, seguindo a sintaxe: tipo nome;. Por exemplo: int x;, String nome;, boolean ligado;, etc. As variáveis podem ser inicializadas com um valor, seguindo a sintaxe: tipo nome = valor;. Por exemplo: int x = 10;, String nome = "João";, boolean ligado = true;, etc. As variáveis podem ser modificadas com uma atribuição, seguindo a sintaxe: nome = valor;. Por exemplo: x = 20;, nome = "Maria";, ligado = false;, etc.

Operadores: em Java, os operadores são símbolos que realizam operações sobre os valores ou as variáveis. Os operadores podem ser classificados em aritméticos, relacionais, lógicos, etc. Os operadores aritméticos realizam operações matemáticas, como soma, subtração, multiplicação, divisão, etc. Por exemplo: x + y, x - y, x * y, x / y, etc. Os operadores relacionais realizam comparações entre os valores ou as variáveis, retornando um valor booleano. Por exemplo: x == y, x != y, x < y, x > y, etc. Os operadores lógicos realizam operações booleanas, como conjunção, disjunção, negação, etc. Por exemplo: x && y, x || y, !x, etc.

Estruturas de condições: em Java, as estruturas de condições são blocos de código que executam uma ação ou outra, dependendo de uma condição. As estruturas de condições mais comuns são o if e o switch. O if segue a sintaxe: if (condição) {ação1;} else {ação2;}. Por exemplo: if (x > y) {System.out.println("x é maior que y");} else {System.out.println("x é menor ou igual a y");}. O switch segue a sintaxe: switch (variável) {case valor1: ação1; break; case valor2: ação2; break; ... default: açãoN; break;}. Por exemplo: switch (dia) {case 1: System.out.println("Domingo"); break; case 2: System.out.println("Segunda"); break; ... default: System.out.println("Dia inválido"); break;}.

Estruturas de repetições: em Java, as estruturas de repetições são blocos de código que executam uma ação repetidamente, enquanto uma condição for verdadeira. As estruturas de repetições mais comuns são o while, o do-while e o for. O while segue a sintaxe: while (condição) {ação;}. Por exemplo: while (x < 10) {System.out.println(x); x++;}. O do-while segue a sintaxe: do {ação;} while (condição);. Por exemplo: do {System.out.println(x); x++;} while (x < 10);. O for segue a sintaxe: for (inicialização; condição; incremento) {ação;}. Por exemplo: for (int i = 0; i < 10; i++) {System.out.println(i);}.

Métodos básicos em Java: em Java, os métodos são declarados com um modificador de acesso, um tipo de retorno, um nome e uma lista de parâmetros, seguindo a sintaxe: modificador tipo nome (tipo1 parâmetro1, tipo2 parâmetro2, ...) {corpo;}. Por exemplo: public int soma (int x, int y) {return x + y;}. Os métodos podem ser chamados com um nome e uma lista de argumentos, seguindo a sintaxe: nome (argumento1, argumento2, ...);. Por exemplo: soma (10, 20);. Alguns métodos básicos em Java são:

System.out.println(): imprime uma mensagem na tela, seguida de uma quebra de linha.

 System.out.print(): imprime uma mensagem na tela, sem quebra de linha.

 System.in.read(): lê um byte da entrada padrão.

 Scanner sc = new Scanner(System.in): cria um objeto Scanner que pode ler dados da entrada padrão.

 sc.nextInt(): lê um inteiro da entrada padrão.

 sc.nextDouble(): lê um número real da entrada padrão.

sc.nextLine(): lê uma linha de texto da entrada padrão.

 sc.close(): fecha o objeto Scanner.

Math.abs(): retorna o valor absoluto de um número.

 Math.sqrt(): retorna a raiz quadrada de um número.Math.pow(): retorna a potência de um número elevado a outro.

 Math.sin(): retorna o seno de um ângulo em radianos.

 Math.cos(): retorna o cosseno de um ângulo em radianos.

Math.tan(): retorna a tangente de um ângulo em radianos.

Math.PI: retorna o valor de pi.

Math.E: retorna o valor de e.

String s = new String(): cria um objeto String que armazena uma sequência de caracteres.

s.length(): retorna o comprimento da string
